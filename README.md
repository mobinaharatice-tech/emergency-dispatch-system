# 🚑 Emergency Dispatch System in C

A console-based Emergency Dispatch System implemented in C for managing emergency incidents, dispatching response units, and simulating emergency operations.

## 📌 Overview

This project simulates the workflow of an emergency response center. It allows users to create incidents, manage emergency units, dispatch responders, and analyze system statistics through an interactive menu.

The project was developed in C using modular programming principles and custom data structures.

---

## ✨ Features

### 🚨 Incident Management
- Create emergency incidents
- Store incident information
- Track incident status
- Manage incident priorities

### 🚒 Unit Management
- Register emergency units
- Different unit types
- Assign units to incidents
- Update unit availability

### 🗺️ Map & Dispatch
- Dispatch nearest available units
- Manage emergency locations
- Route simulation

### 📊 Statistics
- Incident statistics
- Unit utilization
- Dispatch reports
- Simulation summaries

### ⚙️ Configuration
- Load configuration from file
- Adjustable simulation parameters
